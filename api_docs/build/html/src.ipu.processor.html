src.ipu.processor.image.image_to_neuron_list(image, polarize=False)¶

Plays the role of Ganglionic Layer in retina.

Responsible for converting visual stimuli to neuronal activity so it can be passed along the cortical pathways Input: Standard Image Data Output: List of neurons from various Thalamic Visual layers (LGN) that are activated

Output format:
neuron_list =
{

“cortical_area_1” : {list of activated neurons}, “cortical_area_2” : {list of activated neurons}, “cortical_area_3” : {list of activated neurons}

}
