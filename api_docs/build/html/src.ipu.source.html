src.ipu.source.folder_monitor module¶

Source: https://camcairns.github.io/python/2017/09/06/python_watchdog_jobs_queue.html

This class inherits from the Watchdog PatternMatchingEventHandler class. In this code our watchdog will only be triggered if a file is moved to have a .trigger extension. Once triggered the watchdog places the event object on the queue, ready to be picked up by the worker thread
