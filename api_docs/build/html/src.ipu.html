src.ipu.fcl_injector module¶

This module plays the role of the information gateway to the cortical neurons. After physical stimuli is processed by the Input Processing Unit (IPU), it will be translated to neuronal activation and fed to Fire Candidate List (FCL).

Expectations: - To receive the list of activated IPU neurons from the corresponding IPU module and inject them into FCL - To monitor flow of information and act as needed

src.ipu.ipu_controller.initialize()¶

This function will monitor the IPU folder and other possible input devices such as a camera or mic for data and pass them along to the IPU module to have them converted to neuronal activity that in turn can be passed to cortical areas via FCL injection.
